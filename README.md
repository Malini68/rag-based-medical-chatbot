# Multimodal Medical Chatbot using RAG

This project aims to develop a multimodal medical chatbot that provides
preliminary diagnosis using text and image inputs through
Retrieval-Augmented Generation (RAG).

## Technologies
- Python
- Streamlit
- LangChain
- FAISS
- Gemini API

## Current Status
Zeroth review prototype with basic structure and setup completed.
